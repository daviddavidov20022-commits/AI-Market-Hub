## 2. 🧱 КЕРАМИЧЕСКАЯ ПЛИТКА — Ответ

### Да, это реально!

**Технологии:**
1. **Деколирование** — печать на специальной переводной бумаге → обжиг в печи 800-1200°C. Самый долговечный способ.
2. **UV-печать на плитке** — прямая печать UV-чернилами. Не требует обжига, но менее стойко к истиранию.
3. **Сублимация на специальной плитке** — с полимерным покрытием. Для декоративных целей (панно, фартук без прямого контакта с водой).

### Варианты применения:
| Применение | Технология | Подходит? |
|-----------|-----------|-----------|
| Кухонный фартук | UV-печать + защитный лак / закалённое стекло поверх | ✅ Да |
| Ванная (стены) | Деколь + обжиг | ✅ Да, но дорого |
| Ванная (пол) | ❌ Не рекомендуется | Истирание |
| Декоративное панно | Сублимация / UV | ✅ Отлично |
| Столешница | UV + эпоксидное покрытие | ✅ Возможно |

### Реальность для WB/Ozon:
- Тяжёлый и хрупкий товар → дорогая логистика
- Высокий % брака при доставке
- **Рекомендация**: Продавать как **декоративное панно из 1-4 плиток** (подарочный формат), НЕ как строительный материал

> **💡 СОВЕТ:** Лучшая альтернатива для фартука — **стеклянный скинали** с AI-принтом. Легче, безопаснее, тоже продаётся на WB. UV-печать на закалённом стекле.

## 3. МОДЕЛИ РАБОТЫ — Плюсы и Минусы

### Модель 1: Свой аккаунт на WB (FBS — Fulfillment by Seller)

```
Клиент заказывает в TG → Ты покупаешь заготовку у поставщика →
Поставщик наносит принт → Отправляет на склад WB → WB доставляет
```

**✅ Плюсы:**
- Полный контроль цен — ты сам ставишь цену, маржа максимальная
- Свой бренд — можешь строить бренд, карточки товаров твои
- Аналитика — полный доступ к статистике WB
- Масштаб — можно расширять ассортимент без ограничений
- Отзывы копятся на ТВОЕЙ карточке

**❌ Минусы:**
- Начальные вложения — нужно покупать товар ДО продажи
- Риски возвратов — кастомный товар, возврат = потеря
- Бухгалтерия — нужен ИП/ООО, налоги, отчётность
- Штрафы WB — за просрочку поставки, отсутствие товара
- Работа с поставщиком — нужно контролировать качество, сроки

### Модель 2: Партнёрская (Дропшиппинг через поставщика)

```
Клиент заказывает в TG → Ты передаёшь заказ поставщику →
Поставщик всё делает сам (печать + склад WB) →
Ты получаешь % с продажи
```

**✅ Плюсы:**
- Ноль вложений — не покупаешь товар, не рискуешь деньгами
- Нет логистики — поставщик сам везёт на склад
- Простой старт — можно запустить за 1 день
- Нет бухгалтерии — юридически ты рекламный канал

**❌ Минусы:**
- Маленькая маржа — 10-20% vs 50-70% при своём аккаунте
- Нет контроля — качество, сроки, упаковка не в твоих руках
- Нет бренда — товар на карточке поставщика
- Зависимость — поставщик может уйти / поднять цены
- Нет отзывов — отзывы идут поставщику, не тебе

### Модель 3: ⭐ ГИБРИДНАЯ (РЕКОМЕНДУЕМАЯ)

```
Свой аккаунт WB + Поставщик-исполнитель (белый label)
```

**Как это работает:**
- Твой аккаунт WB — карточки, бренд, цены твои
- Поставщик = производство — он только печатает и отвозит на склад WB по ТВОЕМУ штрихкоду
- Оплата по факту — платишь поставщику после получения денег от WB
- Масштабирование — можно подключить нескольких поставщиков

**Схема оплаты:**
```
WB продал кружку за 990₽
  → Комиссия WB: ~15% = 149₽
  → Логистика WB: ~80₽
  → Поставщику (себестоимость + печать + доставка на склад): ~250₽
  → ТВОЯ ЧИСТАЯ МАРЖА: ~511₽ (51.6%)
```

> **⚠️ ВАЖНО:** Гибридная модель — лучший вариант для старта. Ты получаешь маржу как при своём аккаунте, но не держишь склад и не занимаешься производством. Нужен только ИП + договор с поставщиком.

```

```

**Техническая реализация:**
1. Пользователь генерирует картинку через AI API

3. Пользователь видит реалистичный превью
4. Может крутить/зумить/двигать изображение на товаре
